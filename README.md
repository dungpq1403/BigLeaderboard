# BigTournament

> A place where you can host your tournaments with many shortcuts to deliver a wonderful experience to users. This web was developed using React Next.js for frontend and Node.js/Express for backend

This project includes:

Backend: 
- **RESTful API**: to support CRUD operations in databases
- **CORS (Cross-Origin Resource Sharing)**: to support API calls from two different ports (http://localhost:3000 (Frontend) call API to http://localhost:5000 (Backend)),
- Store login token in cookies (not yet but will be fixed), verifies token before any important action throughout the process of using the web.
- Using MySQL Workbench databases to manage data throughout the project

Frontend: 
- Auto-generate brackets with just a click of a button.
- Update tournaments, participants in real-time whenever user registers.
- Setting up user profile by fetching API profiles from enka.network to, basically show off your skills in that specific game.
- Pre-filled button to make the process of registration for a tournament even faster.
